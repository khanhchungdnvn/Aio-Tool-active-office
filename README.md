Kích hoạt phần mềm
Để có thể kích hoạt phần mềm Office 2010 , bạn cần tải thêm phần mềm thứ 3 AIO Tool. Download thành công, bạn giải nén file bằng WinRAR. Sau đó, bạn tìm đến file có tên Activate AIO rồi khởi chạy bằng quyền “Run as administrator”.
